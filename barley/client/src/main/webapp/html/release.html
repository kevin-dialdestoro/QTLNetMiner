Hordeum vulgare

Barley Release Notes

The Barley (Hordeum vulgare) information network contains:

The network contains 79,379 barley genes (from MIPS and IPK) of which 26,159 are high and 53,220 are low confidence. Note that 2/3 of the barley genes have been mapped to chromosomes 1-7 and the remaining ones are assigned to chromosome 0.

    Datasets used for integration

    • Gene
      • HighConf genes ProteinSeq.fa
      • LowConf genes ProteinSeq.fa
    • Protein
      • Gff3 File IPK Gatersleben
    • Protein-Domain
      • BioMart InterPro Protein Domains and HighConf genes HumReadDesc
      • BioMart PFam Protein Domains and HighConf genes PFAM
    • Gene Ontology
      • HighConf genes GOs
    • References
      • Arabidopsis (TAIR) see Release notes for Arabidopsis
      • Plants (UniProt) - Information of well observed plant organisms excluding Arabidopsis
        UniProt: taxonomy:33090 NOT taxonomy:3702 AND reviewed:yes
    • Homology
      • BioMart Homologs Barley - Arabidopsis
      • Decypher
    • Ontologies
      • gene_ontology.1_2.obo
      • interpro2go
      • pfam2go
      • ec2go
